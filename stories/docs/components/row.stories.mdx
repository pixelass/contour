# Row

Row is a sublevel component. It can be used multiple times in a `<Column/>`.

```jsx
import { Column, Grid, Row } from "@contour/react";

export default function App() {
	return (
		<Grid>
			<Column>
				<Row>{/* ... */}</Row>
				<Row>{/* ... */}</Row>
			</Column>
		</Grid>
	);
}
```

## Row Properties

| Name       | Inherit | Required | Default | Description                                       |
| :--------- | :------ | :------- | :------ | ------------------------------------------------- |
| `colCount` | No      | No       | -       | Defines the number of columns for each breakpoint |

## Additional Properties

| Name             | Required | Default | Description                                      |
| :--------------- | :------- | :------ | :----------------------------------------------- |
| `as`             | No       | -       | The Component or Element that should be rendered |
| `strategy`       | No       | `flex`  | One of `flex`, `grid`                            |
| `align`          | No       | -       | One of `start`, `center`, `end`                  |
| `justifyContent` | No       | -       | One of `start`, `center`, `end`                  |

## Strategy: Grid

You can choose one of two strategies: `flex` (default) or `grid`

When chosing `grid`, `<Row/>` becomes a standalone layout.

```jsx
import { Column, Grid, Row } from "@contour/react";

export default function App() {
	return (
		<Grid>
			<Column>
				<Row strategy="grid">{/* ... */}</Row>
				<Row>{/* ... */}</Row>
			</Column>
		</Grid>
	);
}
```

<Grid>
	<Column>
		<Row strategy="grid" align="start">
			<Rainbow length={12} />
		</Row>
		<Row>
			<Rainbow length={12} />
		</Row>
	</Column>
</Grid>

## Compound Row

To use a different grid logic for a section, you can adjust the `<Row/>`.

```jsx
import { Column, Grid, Row } from "@contour/react";

export default function App() {
	return (
		<Grid>
			<Column colSpan={{ l: 9 }}>
				<Row colCount={{ l: 4, xl: 4 }} justifyContent="center">
					{/* ... */}
				</Row>
			</Column>
		</Grid>
	);
}
```

<Grid>
	<Rainbow length={12} />
	<Column colSpan={{ l: 9 }}>
		<Row colCount={{ l: 4, xl: 4 }} justifyContent="center">
			<Rainbow length={3} />
		</Row>
	</Column>
	<Rainbow length={15} />
</Grid>
